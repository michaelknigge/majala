Welcome

majala is a professional exe-wrapper for Java applications written in C#. The use of C# and the .NET Framework makes it possible to create a nearly 100% non-native Windows Java application - no chance for buffer overflows or similar risks that may come up when using exe-wrappers written in C or C++.


majala supports various kinds of application types, like console applications, graphical applications and Windows services, all as 32 or 64 bit processes.


Due to the fact that the .NET Framework is nearly installed on every modern Windows system, majala should run without any problems on every system.
